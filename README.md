### Motor Imagery BCI Classifier
EEG-based brain-computer interface using Common Spatial Patterns and machine learning for motor imagery classification. Real-time signal processing pipeline with artifact rejection and cross-subject validation.

### P300 Speller System
Event-related potential detection for assistive communication. Row-column paradigm with ensemble classification for robust character selection and optimized Information Transfer Rate.

### Real-time Neural Decoder
Kalman filter and LSTM-based decoders for movement prediction from neural signals. Closed-loop simulation demonstrating real-time cursor control with sub-100ms latency.
